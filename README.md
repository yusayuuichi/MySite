# ResumeSite
Created with CodeSandbox
建造屬於自己的履歷網站

# 目的
- 學習 React & Hooks 的基礎使用
- 搭配後端使用 Node.js 實現CRUD 
- 履歷兼作品集

# 使用套件
- eslint
- prettier

# Node.js版本
v16.10.0
```
nvm install v16.10.0
```

# 啟動 Server
```
yarn
yarn run start
```
